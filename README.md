# json-resume
Inspired from https://jsonresume.org/

# Getting Started
Run install
```
npm install
```
Create template resume
```
npm run resume:init
```
To export
```
npm run resume:export <filename.(html|pdf)>
```
Options
- `--theme <theme>` kendall, flat, short, stackoverflow
